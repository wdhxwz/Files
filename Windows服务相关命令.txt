【安装命令】sc create "serverName" start= auto binPath= "可执行文件路径，包括相关启动参数" DisplayName= "服务显示名称"
【添加服务描述】sc description "服务名称" "服务描述"
【启动命令】sc start "服务名称"
【停止命令】sc stop "服务名称"
【卸载命令】sc delete "服务名称"
